About Brief history, browser support, and more

History

The _UT library was born out of the need to meet complex web analytics requirements.

While working primarily with Omniture Sitecatalyst, a JavaScript script file was necessary to handle the business logic. However, as the requirements became increasingly complex, someitmes involving multiple solutions, it became evident that the javsacript should be refactored into a library to manage the implementation. Thus was born the _UT library.

Browser support

_UT is tested and supported in major modern browsers like Chrome, Safari, Internet Explorer, and Firefox. Although more testing is required.

  • Latest Safari
  • Latest Google Chrome
  • Firefox 4+
  • Internet Explorer 7+
  • Opera 11
